Desenvolvedor em Desenvolvimento 🕩 🖖 🤖

A ideia é sempre desenvolver todos os dias , nessa fase estou a melhorar o meu desenvolvimento voltado mais para web, melhorando como desenvolvedor web a minha linguagem de eleição aquela que eu tenho como primeira opção para resolver problemas é o c# mas como desenvolvedor em desenvolvimento eu não me privo apenas em uma única linguagem eu procuro sempre ver, entender e dominar outras matérias de preferência algo que agrege uma funcionalidade no meu desenvolvimento.

Ser um desenvolvedor é uma rotina que obriga a envolver se com outras tecnologias que no final servem para compor o nosso proceso de desenvolvimento de sistema , para trabalhar com desenvolvimento de sites é necessário aprender a trabalhar com <html> que é a linguagem padrão para formatação de sites como recurso complementar é necessário também aprender atrabalhar com css que serve basicamente para estilizar o nosso site . O site em si é um recurso estático sem interatividade nenhuma é aí onde entram as linguagens para dinamismo de sites, o mais comum e atulizado globalmente é o javascript mas também podemos dar iteratividade em um site usando a linguagem C# em um projecto Blazor que por acaso é a minha linguagem de eleição.
