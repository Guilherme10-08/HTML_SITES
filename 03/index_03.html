Sobre C#

C# é uma linguagem de Programação da Microsoft que roda por cima da plataforma dotnet uma linguagem fortemente tipada e orientada a objecto, linguagem de alto nível compilada e com propósito geral de desenvolvimento , usando o C# como linguagem de desenvolvimento nós podemos desenvolver diversos e variados sistemas como aplicações DeskTop,Web,Mobile e Game.

  • Possibilidades de Deenvolvimento com a plataforma dotnet/c#

DeskTop
Utilizando o WindowsForm
WEBSITES/APIs
Utilizando o Aspnet especificamente as suas derivadas, Blazor e RazorPage
MOBILE
Utilizando o MAUI
GAME
Utilizando o XAMARIM
